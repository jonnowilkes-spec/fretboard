# fretboard

Fretboard Explorer Tool.

This is a simple browser-based tool for visualising notes, scales, chords and intervals across a guitar fretboard. It runs entirely in the browser and requires no installation.

Features

Select different tunings

Highlight scales and modes

Show intervals or note names

Toggle sharps/flats

Show or hide individual strings

Lightweight and mobile-friendly

How to Use
1. Choose a tuning

Use the tuning dropdown to select the instrument tuning you want (standard, drop tunings, custom setups, etc.).
The fretboard updates instantly.

2. Pick a root note

Select a root note. All notes belonging to your chosen scale or mode will highlight on the fretboard.

3. Select a scale or mode

Choose from major, minor, pentatonics, modal scales and more.
The tool displays intervals and note names automatically.

4. Toggle display options

Show/Hide Note Names

Show/Hide Intervals

Colour Theme Selection

String Visibility

These options help you practice patterns or visualise shapes more clearly.

5. Interact with the fretboard

Hover or tap on any note to display its name and interval relative to the selected root.

Requirements

Just a modern browser. No installation, no backend, no user data stored.

Hosting

This tool is a static website and works on GitHub Pages. Just open index.html directly or visit the hosted link.
